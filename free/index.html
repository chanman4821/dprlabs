Inbox-to-Action Agent

Your inbox is full. This helps. Paste a message thread and get back three things you can use in under a minute.

  • A summary — what the thread is really about, in a few bullets.
  • A draft reply — ready to send, in your tone, with blanks marked [like this].
  • An action list — the next steps, who owns each, and what needs you to decide.
Get it on GitHub → New to this? Start here
Runs in your terminal # grab it
git clone https://github.com/chanman4821/inbox-to-action.git
cd inbox-to-action

# see exactly what it sends — no key needed
python inbox_to_action.py examples/sample-thread.txt --dry-run

# add your OpenAI or Anthropic key, then run it for real
python inbox_to_action.py examples/sample-thread.txt

Runs local

It's one Python file on your computer. Nothing installs a server or a background service.

Your key, your bill

Bring your own OpenAI or Anthropic key. You pay the provider directly — cents per thread on the default models.

No telemetry

Your text goes to the AI provider you pick and nowhere else. We never see it. Check the code — it's short.

MIT licensed

Yours to keep, fork, and change. If we disappear tomorrow, it still works.
